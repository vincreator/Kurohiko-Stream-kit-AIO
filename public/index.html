🎭
Custom Deck
Upload gambar, video, atau audio. Atur animasi masuk/keluar, ukuran, posisi, durasi — semua per-meme.
🎮
Deck View
Tampilan trigger button. Klik card untuk langsung trigger meme ke OBS. Kelompokkan per tab, drag & drop untuk reorder.
📱
Connect to Phone
Buka Deck View di HP lewat browser via WiFi yang sama. Trigger meme langsung dari HP tanpa alt-tab.
🖥️
OBS Overlay
Tambahkan Browser Source di OBS dengan URL yang tersedia. Meme langsung muncul di output stream.
🟢
Chroma Key
Upload video green screen — aktifkan Chroma Key di settings meme untuk otomatis hapus background hijau.
⚔️
K/D Counter
Buat counter kill/death atau apapun. Nilai tersimpan ke .txt — tambahkan sebagai Text Source di OBS.
⌨️
Keyboard Shortcut
Setup hotkey untuk trigger meme tanpa klik — langsung dari keyboard saat sedang gaming.
